Historical Military Battles¶

Jemimah E.P. Salvacion

Motivation¶

The "CAA Database of Battles, Version 1990", shortnamed "CDB90", contains information on over 600 battles that were fought between 1600 AD and 1973 AD. When it comes to military tacticts, preparing an attack against an enemy involves considering physical attributes that are out of man's control. My goal is to see if battle outcomes can be predicted based on the effects of six terrain features that may or may not have affected the outcome of specific battles.

from matplotlib.ticker import PercentFormatter
import matplotlib.pyplot as plt
from matplotlib import colors
import pandas as pd
import numpy as np
from plotnine import *

battles_df = pd.read_csv('battles-2.csv')
battles_df = pd.DataFrame({'ID' : battles_df['isqno'], 'War' : battles_df['war'],
                           'Level_Of_Surprise' : battles_df['surpa'],
                           'Win' : battles_df['wina']})
battles_df = battles_df[battles_df['Level_Of_Surprise'] >= 0]
battles_df = battles_df.dropna()

terrain_df = pd.read_csv('terrain.csv')
terrain_df = pd.DataFrame({'ID' : terrain_df['isqno'], 'Terrain' : terrain_df['terra1'],
                           'Landscape' : terrain_df['terra2']})

weather_df = pd.read_csv('weather.csv')
weather_df = pd.DataFrame({'ID' : weather_df['isqno'], 'Climate' : weather_df['wx1'],
                           'Precipitation' : weather_df['wx2'], 'Temperature' : weather_df['wx3'],
                           'Season' : weather_df['wx4']})

attackers_df = pd.read_csv('battle_dyads.csv')
attackers_df = pd.DataFrame({'ID' : attackers_df['isqno'], 'Aggressor' : attackers_df['attacker']})

main_df = pd.merge(weather_df, terrain_df, on='ID')
main_df = pd.merge(main_df, attackers_df, on='ID')
main_df = pd.merge(main_df, battles_df, on='ID')
main_df = main_df[['ID', 'War', 'Aggressor', 'Climate', 'Precipitation', 'Temperature',
                   'Season', 'Terrain', 'Landscape', 'Level_Of_Surprise', 'Win']]
main_df = main_df.dropna()
